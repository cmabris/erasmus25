# Mejoras Pendientes - Calendario de Eventos

## Vista Mensual del Calendario

**Estado**: Pendiente de mejora

**Problema actual**: La vista mensual actual es funcional pero básica. Utiliza un grid HTML simple que muestra los eventos en cada día del mes.

**Mejora propuesta**: Reemplazar la vista mensual actual por un componente JavaScript moderno de calendario que ofrezca:

- Mejor visualización de eventos múltiples en un día
- Interacciones más fluidas (drag & drop, hover effects)
- Mejor responsive en móviles
- Integración con librerías populares como:
  - **FullCalendar** (https://fullcalendar.io/)
  - **Calendar.js** (https://calendar.js.org/)
  - **Vanilla Calendar** (https://vanilla-calendar.js.org/)
  - O cualquier otra librería moderna de calendarios

**Consideraciones**:
- Mantener la integración con Livewire
- Asegurar que los filtros sigan funcionando
- Mantener la compatibilidad con dark mode
- No romper las vistas semanal y diaria que ya funcionan bien

**Prioridad**: Media-Baja (funcionalidad actual es suficiente, mejora de UX)
