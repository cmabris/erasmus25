# Guía de Usuario - Erasmus+ Centro (Murcia)

Bienvenido a la documentación de usuario de la aplicación **Erasmus+ Centro (Murcia)**, un portal web que centraliza toda la información relacionada con los programas Erasmus+ de Educación Escolar, Formación Profesional y Educación Superior.

---

## Guías Disponibles

### Guía de Administrador

**Archivo:** [guia-administrador.md](guia-administrador.md)

**Destinada a:** Usuarios con roles `super-admin` y `admin`.

**Contenido:**
- Acceso y configuración del sistema
- Dashboard de administración
- Gestión completa de todos los módulos
- Configuración del sistema
- Gestión de usuarios y roles
- Auditoría y logs

---

### Guía de Editor

**Archivo:** [guia-editor.md](guia-editor.md)

**Destinada a:** Usuarios con roles `editor` y `viewer`.

**Contenido:**
- Acceso al sistema
- Creación y edición de contenido
- Flujos de trabajo para solicitar publicación
- Funcionalidades disponibles según el rol

## Roles del Sistema

La aplicación define cuatro roles principales:

| Rol | Descripción | Guía Recomendada |
|-----|-------------|------------------|
| **Super Administrador** | Acceso total al sistema | [Guía de Administrador](guia-administrador.md) |
| **Administrador** | Gestión completa de contenido | [Guía de Administrador](guia-administrador.md) |
| **Editor** | Creación y edición de contenido | [Guía de Editor](guia-editor.md) |
| **Viewer** | Solo lectura | [Guía de Editor](guia-editor.md) |

---

## Requisitos del Sistema

Para utilizar la aplicación correctamente, necesitas:

- **Navegador web moderno**: Chrome, Firefox, Safari o Edge (versiones recientes)
- **Conexión a Internet**: Para acceder al portal
- **Credenciales de acceso**: Proporcionadas por el administrador del sistema

---

## Soporte

Si tienes dudas o problemas con la aplicación:

1. Consulta la guía correspondiente a tu rol
2. Contacta con el administrador del sistema
3. Revisa la sección de preguntas frecuentes (si está disponible)
